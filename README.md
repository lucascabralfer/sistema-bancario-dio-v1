# Projeto de Sistema Bancário em Python V1

Implementação de um sistema bancário simples, seguindo as regras do desafio da DIO.

## Requisitos:

### Operações:
  - Depósito
  - Saque
  - Extrato (mostrar todas as operações e o saldo, caso não tenham sido feitas operações até o momento, avisar ao usuário)

### Exceções:
  - Valores negativos
  - Saques maiores de 500 reais
  - Mais de 03 saques em um mesmo dia
  - Sacar valores maiores que o saldo
